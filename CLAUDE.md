## Design direction

- Light background only
- Warm white, soft beige, cream sections
- Champagne gold accents
- Charcoal typography
- Premium UAE B2B manufacturing feel
- No dark luxury brochure style
- No generic template look
- Spacious layout
- Rounded premium cards
- Soft shadows
- Subtle animation only
- Real factory / food production mood

## Technical direction

- Next.js App Router
- TypeScript
- Tailwind CSS
- Responsive
- SEO metadata
- Static contact form for now
- WhatsApp floating button
- Clean reusable components
- No backend required at this stage

## Important content notes

Do not invent numbers such as production capacity, outlets, factory size, or team size until the client provides verified data.

The website must feel like a B2B manufacturing sales website, not a company profile PDF converted to HTML.
